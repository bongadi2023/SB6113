Git
Students:  Stuents to Git Clone
Create a github account for the practical projects.
Create a repository for every project.
Add the instructor as a collaborator.
Clone the created project to your local machine:
	git clone {link_to_created_repo}
Make all your changes within the project folder.
In order to push changes to the remote repository:
	Add all changes to staging area:
		git add .
	Commit all the changes before pushing:
		git commit -m "message"
	Push changes to the main branch:
		git push origin main

Instructor:
Clone the student's project to your local machine:
	git clone {link_to_remote_repo}
Review all and make comments
Once done, add and commit the changes:
	git add .
	git commit -m "message"
Then push the changes for students to review.
	git push origin main
